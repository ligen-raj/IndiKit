# IndiKit

**IndiKit** is a SwiftUI framework for creating sleek, customizable status indicators — including battery, Wi-Fi, network, and sound levels.  
It’s built for flexibility, animation, and clean integration into any SwiftUI project.

### ✨ Features
- Customizable colors, sizes, and styles
- Battery, Wi-Fi, network, and sound indicators
- SwiftUI-native and lightweight
- Ready for Swift Package Manager integration
- Perfect for dashboards, widgets, or iOS utilities
